🛒 Online Store Web Application

📌 Project Overview

This project is a full-stack e-commerce web application developed using PHP, MySQL, HTML, CSS, and JavaScript. It provides a complete online shopping experience, including user authentication, product browsing, shopping cart management, wishlist functionality, secure checkout, and order tracking.

The system was designed to simulate a real-world online store with both customer and shopping management features.

---

✨ Features

User Authentication

- User Registration (Sign Up)
- User Login
- Password Reset
- Secure authentication system

Product Management

- Browse available products
- View detailed product information
- Product images
- Organized product display

Shopping Cart

- Add products to cart
- Update product quantities
- Remove products from cart
- Cart is available only for authenticated users
- Guests attempting to add items are automatically redirected to the Login page

Wishlist

- Add products to wishlist
- Remove products from wishlist
- Wishlist is available only for logged-in users
- Guests are redirected to the Login page when attempting to use this feature

Checkout System

- Secure checkout process
- Order confirmation
- Order placement

Order Tracking

Users can track the status of their orders, including stages such as:

- Processing
- Shipped
- Delivered

🗄 Database

The project includes the SQL database file required to run the application.

Steps:

1. Import the provided ".sql" file into phpMyAdmin.
2. Start Apache and MySQL.
3. Place the project folder inside the web server directory.
4. Open the project in your browser.

2. Import the SQL database into MySQL.
3. Place the project folder inside your web server directory (e.g. "htdocs" if using XAMPP).
4. Start Apache and MySQL.
5. Open the project in your browser.
